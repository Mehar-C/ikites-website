# Partner logos (The Collaborative marquee)

Add logo images here to show in the "The Collaborative" section. Supported filenames:

- `procore.svg` (or .png)
- `evertech-xr.svg`
- `buildots.svg`
- `perspect-analytics.svg`

Use SVG or PNG; recommended max height ~80px. If a file is missing, the company name is shown as text instead.
